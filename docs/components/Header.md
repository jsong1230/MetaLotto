# Header 컴포넌트

## 개요
애플리케이션 헤더 컴포넌트입니다. 로고, 네비게이션 메뉴, 지갑 연결 버튼을 포함하며 반응형 디자인으로 데스크톱과 모바일 환경을 모두 지원합니다.

## 파일 경로
`/Users/jsong/dev/metadium-github/MetaLotto/frontend/src/components/Header.tsx`

## Props
이 컴포넌트는 props를 받지 않습니다.

## 데이터 소스
- `usePathname`: 현재 경로 (네비게이션 활성 상태 결정)

## UI 구성

### 데스크톱 헤더
```
┌─────────────────────────────────────────────────────┐
│ [로고]        [홈] [히스토리] [내 티켓]  [지갑] │
└─────────────────────────────────────────────────────┘
```

### 모바일 헤더
```
┌─────────────────────────────────────────────────────┐
│ [로고]                                    [지갑] │
├─────────────────────────────────────────────────────┤
│   [🏠]  [📜]  [🎫]                    │
│   홈    히스토리 내 티켓                         │
└─────────────────────────────────────────────────────┘
```

## 기능

### 로고
- "M" 로고와 "MetaLotto" 텍스트
- 인디고-퍼플 그라디언트 배경
- 둥근 사각형 형태
- 홈(`/`)으로 링크

### 네비게이션 메뉴
| 메뉴 | 경로 | 아이콘 |
|------|------|--------|
| 홈 | `/` | 🏠 (집) |
| 히스토리 | `/history` | 📜 (시계) |
| 내 티켓 | `/my-tickets` | 🎫 (티켓) |

#### 활성 상태 스타일
- 인디고 배경 (light) / 인디고 다크 배경 (dark mode)
- 인디고 텍스트

#### 비활성 상태 스타일
- 회색 텍스트
- 호버 시 회색 배경

### 모바일 네비게이션
- 하단 탭 형태
- 아이콘 + 텍스트
- 상단 경계선으로 구분
- 데스크톱에서 숨김 (`hidden md:flex`)

### 지갑 연결 버튼
- `WalletConnectButton` 컴포넌트 사용
- 우측 정렬

## 반응형 브레이크포인트
- 모바일: 기본 (하단 탭)
- 태블릿: 768px 이상 (상단 메뉴)
- 데스크톱: 1024px 이상 (상단 메뉴)

## 스타일링
- Tailwind CSS 사용
- 다크 모드 지원
- 고정 위치 (`sticky top-0`)
- z-index: 50 (다른 콘텐츠 위에 표시)
- 흰색/다크 그레이 배경
- 하단 경계선

## 사용 예시
```tsx
import { Header } from '@/components/Header';

export function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html>
      <body>
        <Header />
        <main>{children}</main>
      </body>
    </html>
  );
}
```

## 의존성
- next/navigation (`usePathname`, `Link`)
- @/components/WalletConnectButton

## 참고
- 이 컴포넌트는 `'use client'` 지시어를 사용하여 클라이언트 컴포넌트로 작동합니다.
- 네비게이션 활성 상태는 `usePathname` 훅으로 결정합니다.
- 모바일 네비게이션은 데스크톱에서 숨깁니다.
